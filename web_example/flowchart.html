ML-Based Network Security System

머신러닝 기반 네트워크 공격 방어 시스템 아키텍처

1
머신러닝 학습 모델
다양한 네트워크 공격 데이터를 활용하여 머신러닝 모델을 학습시키고, 학습된 모델을 기반으로 공격 탐지 및 방어 시스템을 구축합니다.
Python TensorFlow/PyTorch Scikit-learn
주요 기능
• 네트워크 트래픽 패턴 분석
• 이상 탐지 알고리즘
• 공격 유형 분류
2
웹 인터페이스
공격 시나리오 선택 및 SDN 자동방어 옵션을 제공하는 웹 인터페이스를 통해 사용자가 시뮬레이션을 제어하고 공격을 시작할 수 있습니다.
Node.js Express.js React/Vue.js
JSON 데이터 전송
제공 기능
• 공격 시나리오 선택
• SDN 방어 설정
• 실시간 모니터링
3
네트워크 시뮬레이션
웹에서 전송받은 JSON 데이터를 기반으로 Mininet을 활용하여 실제 네트워크 환경을 시뮬레이션합니다.
Python Mininet OpenFlow
시뮬레이션 환경
• 가상 네트워크 토폴로지
• SDN 컨트롤러 연동
• 트래픽 생성 및 분석
4
고성능 패킷 처리
성능이 중요한 패킷 파싱과 커널 레벨 모듈을 C/C++로 구현하여 실시간 고속 패킷 처리를 담당합니다.
C/C++ DPDK eBPF
최적화 영역
• 빠른 패킷 파싱
• 커널 레벨 필터링
• 하드웨어 가속
5
결과 데이터 처리
시뮬레이션과 패킷 처리 결과를 JSON 형태로 통합하여 웹 인터페이스로 전송할 수 있는 형태로 가공합니다.
Python JSON REST API
결과 JSON 출력
출력 데이터
• 공격 탐지 결과
• 네트워크 성능 메트릭
• 방어 효과 분석
6
웹 UI 시각화
처리된 결과 JSON 데이터를 받아와서 직관적이고 전문적인 웹 UI를 통해 사용자에게 시각화하여 제공합니다.
Node.js Chart.js/D3.js WebSocket
시각화 요소
• 실시간 대시보드
• 공격 패턴 그래프
• 네트워크 토폴로지 뷰
